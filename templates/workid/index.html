Reclutador IA para Work-ID

Este sistema de reclutamiento utiliza inteligencia artificial para identificar y recomendar al mejor candidato para un cargo específico. Analiza una base de datos con información detallada de miles de usuarios, filtrando candidatos en función del cargo, comuna y experiencia proporcionados por el usuario. Si no se encuentran coincidencias exactas, el sistema consulta a la IA para sugerir comunas cercanas y encontrar al candidato más adecuado. La IA evalúa factores clave como el último cargo ocupado, la experiencia acumulada, los estudios realizados y la ubicación, para proporcionar una recomendación precisa y alineada con las necesidades del puesto solicitado.
